# Cabinet installation

1. Install the Arcadium release, supported emulators and ROM/media storage.
2. Copy JSON examples from `config/` to local JSON configuration files.
3. Set local ROM, media and emulator paths through maintenance mode.
4. Scan the library.
5. Configure autostart for the dedicated arcade user.
6. Test controller navigation, game launch, emulator exit and return to Arcadium.

Do not commit local JSON configuration, database files, ROMs or artwork libraries.
